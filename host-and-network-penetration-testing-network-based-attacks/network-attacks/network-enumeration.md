# Network Enumeration

<figure><img src="../../.gitbook/assets/image (383).png" alt=""><figcaption></figcaption></figure>

Ci concentriamo sulle porte aperte per cercare di imparare il più possibile sui servizi running su ognuna delle porte. Approfondiamo ogni servizio attivo.

<figure><img src="../../.gitbook/assets/image (384).png" alt=""><figcaption></figcaption></figure>

L'Enumeration è un processo attivo perché andiamo ad interagire con i sistemi target, nello specifico con le i servizi sulle porte aperte dei sistemi target. Al contrario l'information gathering è passivo perché usiamo informazioni già disponibili liberamente.

Con l'enumeration andiamo ad interagire e raccogliere informazioni sugli specifici servizi attivi, non ci limitiamo a fare uno scan delle porte come nell'information gathering.
