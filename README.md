# Information Gathering

Cercare info su un target, azienda o persone

* Passive: Trovare più info possibili senza attivamente ingaggiare il target
* Active: Trovare più info possibili ingaggiando attivamente il target

### Passive Information Gathering

* IP e info sul DNS
* domain names e info sulla domain ownership
* email e profili social
* web technologies usate su siti target
* subdomains

### Active Information Gathering

* Trovare porte aperte
* Comprendere l'infrastruttura interna di una organizzazione o rete target
* Enumerare informazioni sul sistema target
