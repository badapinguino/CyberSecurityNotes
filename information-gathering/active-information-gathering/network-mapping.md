# Network Mapping

Trovare host attivi e relative porte aperte, nella rete.

Consiste nell'identificare dispositivi, host e elementi dell'infrastruttura di rete della rete target.

Serve per avere informazioni sul layout di rete, e identificare potenziali entry point.

Obiettivi:

* Live hosts
* Open ports e servizi
* network topology mapping: network, router, switch, firewall
* OS Fingerprinting
* Service Version detection
* Identificare filtri e misure di sicurezza: firewall, IPS, e altre misure di sicurezza, per capire quali sono le difese della rete.

Tool utilizzato: Nmap (Network Map)
